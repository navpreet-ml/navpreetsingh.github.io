Autonomous Driving – Car detection (Deep Learning)

  • Using a dataset from drive.ai (video taken from a car-mounted camera), implemented an object detection algorithm based on YOLO Model.
  • Input images were of size 608 x 608 x 3 and using a grid of size 19 x 19, a list of bounding of size (pc, bx, by, bh, bw, c) where c was an 80 dimensional vector representing 80 different object classes. 5 different anchor boxes were drawn by taking a max across different classes and making predictions with a threshold of 0.4.
  • A non-maximum suppression (NMS) was used to avoid overlapping boxes and select the one with the max IoU.
